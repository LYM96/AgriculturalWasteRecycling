# AgriculturalWasteRecycling
农业废弃物回收系统的设计与实现

三端用户：管理员、农户、企业用户

本系统是基于SpringBoot+Vue的农业废弃物回收系统的设计与实现，本系统操作后端技术为Java语言，后端框架为SpringBoot,前端技术为Vue，数据库为MySQL，数据库工具采用Navicat，开发工具为IDEA、VS Code。以此搭建开发环境实现农业废弃物回收系统的功能。
本系统的功能模块主要有：废弃物发布、废弃物分类、废弃物查询、废弃物识别、评价反馈、客服与帮助、回收订单追踪、农户管理、回收企业管理、回收订单管理、废弃物信息统计、政策与资讯资源模块、系统管理、数据分析与报表。
该农业废弃物回收系统预期有如下功能：
农户：
（1）废弃物查询：按废弃物的种类进行分类，便于用户查找。
（2）政策与资讯资源模块：展示政府部门发布的农业废弃物回收相关的政策文件以及最新的资讯和技术指导，提供废弃物分类和处理技术相关的知识。
（3）废弃物发布: 农户可以发布废弃物的类型、数量、位置等信息。
（4）回收订单追踪：查看订单状况（待接单、已接单、运输中、已完成）。
（5）废弃物识别：利用第三方接口识别废弃物的类型。
（6）回收订单评价：关于好或不好的评价反馈给后台。
（7）回收订单结算：订单回收完毕之后，农户账号收到相应的环保积分。
（8）意见反馈：农户可以通过网站反馈意见。
企业管理员：
（1）农户管理：管理员可查看农户信息，进行增删改查。
（2）回收企业管理：回收企业可以维护自己的基本信息，查看历史回收记录和统计。
（3）回收订单管理：对农户发布的废弃物自动生成订单并对订单状态进行追踪。
（4）数据分析与报表：通过图表展示废弃物的回收量、回收种类等并可以生成废弃物回收的统计数据，供政府部门参考。
（5）系统管理：管理不同角色的权限，设置系统参数等。
